Awards and Certificates

Wondering how I got the certificate?

In 2021 by the end of the academic year, Rosebank college hosted a Hackathon that promised a
R10 000 voucher to a team that can create a solution that can solve any student related problem.
The idea was for participants to research on student related problems or to look around them and
notice what problems they encountered and how they can be resolved.

I created my solution which I Called 'HERO- the student companion'.Hero was supposed to be as the name suggests
a companion for students. I saw that students including myself needed a helping hand in structuring their time,
pacing their studies and keeping track of assessment schedules to increase their chances of success.


They did not have a proper way to measure and track their progress with their studies, so I created some structure for students,
something that will provide the structure they need to excel, and that's where the development of hero came to life.
I programmed the application using ASP. Net framework for the user interface and leveraged the power of C# to write the algorithm.
To store the user input i utilised Microsoft SQL Server to create a database where all the input will be stored.

Feel free to check out my Github repository for the source code or view the Youtube video for the app in the Projects section.
These are the tools I used to create the software that won me third place!!!
