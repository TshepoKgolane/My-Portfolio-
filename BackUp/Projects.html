Mark My Spot - Navigation App!!

In 2021, as part of an Assessment. I had to create a mobile application which was to use a google maps API to locate the user's location, and use that location to show the User any local landmarks around their area

I created the mobile application in Android studio and programmed the logic of the app in Java. I Utilised Firebase to store the values in a database and used JSON to manipulate the data

The application was really challenging as it required many elements to function. Also, the Google Maps API required a subscription which our institution did not provide. To combat this however, I created an account and signed up for a free trial which allowed me to get the API key required to access the API which then allowed me to complete my assignment

The Video to the app can be found on the card to the left, click on the Youtube icon to access it.

That Weaving Company - Web application!!

During my Final year of study 2022. My team was tasked with the function Of developing a web and mobile application for a startup company called 'That Weaving Company' that sells industrial weaving belts.The company required a website where they can sell they're industrial belts online to their customers.

They also required a mobile application that will allow them to manage the material they use to manufacture the products. They needed an App that will tell them how much resources were used, how much is left, and how much was generated.

The web application was built by Visual Studio and C# was used to create the application's functionality, with ASP.NET to build the web pages that allowed users to interact with the application's features.

This web application unfortunately, did not contain a database as during that time, It was not needed.

The TWC Mobile application

That Weaving Company - Mobile application!!

As 'That Weaving Company' also required an application that will allow them to manage the resources they use, the TWC mobile application provided a clean user-friendly mobile application that was easy to use.

The mobile application had the following features:
  • Login/register - validate the user
  • Admin - consists of administrative actions
  • Inventory - Displays the current Inventory
  • Resources - Consists of Resource management actions
  • Products - Displays the overall products manufactured
  • Profile - Displays the Signed In User's details


The App was built in Android Studio using Java as the development language, and Firebase was used as the database to store and retrieve all the data required
